# vue-admin-template

# 本项目基于 vue2.0 搭配 Element-UI 搭建的管理后台系统

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```

```

### Lints and fixes files

```
npm run lint
```
